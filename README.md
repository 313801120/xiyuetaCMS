xiyuetaCMS开源网站管理系统 v1.1.0<br>
基于Asp+Access/SQL Server架构的智能系统，集电脑版、手机版、平板版于一体，无缝切换

<br>
admin 目录为后台管理 <br>
data 目录为access数据库存放<br>
exam 目录为在线考试<br>
inc 目录为程序函数库<br>
install 目录为创建access/mssql数据库，需要先登录，登录密码为 xiyueta.com<br>
tools 目录为工具<br>
UploadFiles 目录为上传图片文件<br>
user 目录为用户管理<br>
web 目录为测试网站<br>
